# Line Election Api

### How to run the project ###

1. Run docker-compose for start database sql server.
- `docker-compose up` or `docker-compose up -d`
2. Start project.
- Linux or macOS `./gradlew bootrun`
- Windows `gradlew.bat bootRun`
